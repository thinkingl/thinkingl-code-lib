Core Package

The following is included in the Core Package of XML for <SCRIPT>:

  • The DOM and SAX parsers
  • The Extra Tools (xmlEscape and xmlIO)
  • The Contributed Add-Ons (including the XPath Contributed Add-on)
  • The souce code for all the pre-built server-side proxies
  • A complete copy of this website (including the documentation, test suites and sample applications)

Unless you specifically need or want to download the Java Proxy Reference Implementation for testing or development, you'll want to download the Core Package of XML for <SCRIPT>.
