Java Proxy Reference Implementation

To allow for easy demonstrations of the server-side proxy capabilities of XML for <SCRIPT>, a Java Proxy Reference Distribution has been developed that includes a pre-configued version of the Apache Tomcat Java web server. This pre-configured web server contains a version of the Java Servlet reference proxy that is configured to run the xmlIOProxy test suite.

The Java Proxy Reference Implementation is also useful for developers who are developing their own server-side proxies to compare the results of thier proxies against a reference implmentation.

You may find more information about XML for <SCRIPT>'s available proxies and the Java Proxy Reference Implementation in the "Documentation" -> "Server-Side Proxies" section of the web site.
