Contact Manager Sample Application (W3C DOM Parser)

This sample application is an implementation of the XML for <SCRIPT> Contact Manager Sample Application using XML for <SCRIPT>'s W3C DOM Parser. It demonstrates how developers can access the W3C DOM Parser directly to build real-world applications.
